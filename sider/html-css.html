HTML & CSS Projekt

I dette projekt skulle jeg lave en hjemmeside med en forside og mindst 3 undersider. Dette projekt gik også ud på at lære og forstår HTML & CSS. Det kan godt siges at dette proejkt var kickstarten til mit viden omkring de 2 programmingsprog. Hvilekt senere hen har hjulpet mig med andre projekt som f.eks. denne hjemmeside. Hver side jeg har lavet skulle også have, hver deres egne titler og minimum et billede. Det program jeg brugte til at skrive html & css koderne i er code editoren Atom. Atom gjorte det nemt at skille html og css koderne fra hinanden altså sætte dem side om side, hvilket også kan ses i billedet nedefor.

Det jeg også lærte med dette projekt var nok, hvordan jeg stille mapper op ordentlig og overskueligt så det ikke bare noget rod når der skal vises noget frem. Og ikke nok med at sætter mapper og filer op korekt, så lærte jeg også at sætte koderne op så det ikke bare står hulter til bulter over det hele men faktisk giver mening der hvor koderne er placeret
