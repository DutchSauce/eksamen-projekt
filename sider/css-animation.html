CSS Animation

Til dette projekt skulle jeg lave en animation i css, hvor jeg tog Adobe Illustrator i brug. Jeg startede med at ud forme/tegne som i nok kan se er mine initaler "AHD". Ved hjælp af CSS, ligner det at mine initaler bliver skrevet ned på hjemmesiden.

Dette projekt har virkligt givet mig en meget bredder viden for hvad man kan i css og hvordan man sætter kode op så man kan justere farten på ens animation.
